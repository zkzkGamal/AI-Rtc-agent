# Agent: AI Call Agent Engine

The intelligent brain of the AI-RTC-Agent system that handles reasoning, memory, and decision-making.

## 🎯 Purpose

The agent module provides:
- **Base Agent Class** – Foundation for all agent implementations
- **Conversation Memory** – Context tracking across turns
- **System Prompts** – Customizable AI behavior templates
- **Decision Logic** – Reasoning and response generation
- **Intent Recognition** – Understanding user requests
